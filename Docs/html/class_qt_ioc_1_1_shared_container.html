Detailed Description

The SharedContainer class.

The base container class for the object, providing shared-lifetime.

Shared-lifetime: The object lives as long as it is used by someone from the outside. When all strong references to it are removed, it will be destroyed. When requested again, the destroyed object will be re-created via a factory call.
